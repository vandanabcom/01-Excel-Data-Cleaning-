##📊 Excel Data Cleaning Project
##📌 Project Overview

This project demonstrates how to clean messy data in Microsoft Excel before performing analysis. The dataset contains duplicate records, inconsistent text formatting, blank values, and invalid entries.

##🎯 Objective

Prepare a clean and reliable dataset for business analysis by applying Excel data cleaning techniques.

##🛠 Tools Used
Microsoft Excel
##📚 Excel Skills Used
Remove Duplicates
TRIM
PROPER
IF Formula
COUNTIF
Conditional Formatting
Data Validation
##📋 Data Cleaning Process
Removed duplicate records
Standardized text formatting
Removed leading and trailing spaces
Identified valid and invalid records
Highlighted errors using Conditional Formatting
Prepared a clean dataset for further analysis
##📈 Outcome

The cleaned dataset is ready for reporting, dashboard creation, and business analysis.

📂 Dataset

This project uses a synthetic dataset created for learning and portfolio purposes.
